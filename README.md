# 乡村综合治理数智化平台

## 实现的功能

- **多页面数据大屏**：村情村貌、产业发展、安全防治、民生服务、低碳生活多个主题页面切换。
- **三维实景模型**：基于 Cesium + 3D Tiles 加载村庄实景三维模型，叠加天地图卫星影像底图与影像注记（地名/路名）。
- **村界线叠加**：读取 KMZ 文件，在地图上绘制行政村边界线。
- **村情村貌大屏**：社区概况、土地资源、组织建设、基础设施及公服配套等数据面板，含地球首页与三维模型页两种视图。
- **屋顶光伏测算**：在三维模型上选择屋顶，查看发电量、经济测算等图表分析。
- **模型辅助工具**：相机视角控制、坐标拾取、模型整体高度偏移（用于贴合卫星底图）。

## 技术栈

- **Vue 3**（Composition API / `<script setup>`）
- **Vite 5**（开发与构建）
- **CesiumJS**（三维地球 / 3D Tiles / KMZ）
- **ECharts**（图表）
- **天地图 WMTS**（卫星影像底图与注记）

## 如何启动

> 需先安装 Node.js（建议 18+）。

```bash
# 1. 安装依赖
npm install

# 2. 配置环境变量：复制模板并填入实际值
cp .env.example .env
# 然后编辑 .env，填入 3D 模型地址、天地图 token、后端地址

# 3. 启动开发服务器（默认 http://localhost:5173）
npm run dev

# 4. 打包构建 / 本地预览构建产物
npm run build
npm run preview
```

## 注意事项

- **环境变量**：`.env` 不入库，请从 `.env.example` 复制生成；前端变量必须以 `VITE_` 开头；**修改 `.env` 后需重启 dev 服务器**才生效。
- **天地图 token**：需自行到 [console.tianditu.gov.cn](https://console.tianditu.gov.cn) 申请；上线时请在后台配置域名白名单，防止被盗用。
- **限流（429）**：天地图免费 token 有请求频率限制，高峰时偶发个别瓦片加载失败属正常现象，刷新或缓存后会好转。
- **模型与底图对齐**：模型可能与卫星底图存在高度差，可在对应组件中调整 `MODEL_HEIGHT_OFFSET`（单位：米，负值下沉）微调贴合。
- **跨域（CORS）**：3D 模型与 KMZ 资源需服务器开启跨域支持，否则会加载失败（报 CORS / 403）。
